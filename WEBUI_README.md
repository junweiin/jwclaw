# JwClaw Web UI 使用指南

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install flask flask-socketio python-socketio simple-websocket
```

或者使用requirements.txt：

```bash
pip install -r requirements.txt
```

### 2. 启动Web UI

**方式A：使用启动脚本（推荐）**
```bash
webui.bat
```

**方式B：直接运行Python**
```bash
python webui.py
```

### 3. 访问界面

打开浏览器，访问：**http://localhost:5000**

## ✨ 功能特性

### 1. **实时对话**
- 流式输出，类似ChatGPT的体验

- 清晰显示调用的工具

### 3. **状态监控**
- 显示当前使用的模型
- Skills数量
- 记忆条数
- 连接状态

### 4. **对话管理**
- 清空对话历史
- 自动滚动到最新消息
- 响应式设计，支持移动端

## 🎨 界面说明

### 顶部导航栏
- **JwClaw标志** - 系统名称
- **清空对话按钮** - 清除所有历史记录
- **在线状态** - 显示连接状态

### 状态栏
- **模型** - 当前使用的LLM模型
- **Skills** - 已加载的技能数量
- **记忆** - 记忆系统中的条目数

### 聊天区域
- **用户消息** - 紫色渐变气泡（右侧）
- **助手回复** - 灰色气泡（左侧）
- **思考内容** - 黄色背景，显示Agent的思考过程
- **工具调用** - 蓝色背景，显示调用的工具
- **工具结果** - 绿色背景，显示执行结果

### 输入区域
- **文本框** - 输入消息
- **发送按钮** - 发送消息（或按Enter键）
- **思考指示器** - 显示"JwClaw正在思考..."

## 💡 使用技巧

### 1. **快捷键**
- `Enter` - 发送消息
- `Shift + Enter` - 换行（如果未来支持多行）

### 2. **查看工具执行**
当Agent调用工具时，你会看到：
```
🔧 调用工具: shell(Get-ChildItem C:\Users\jw\Desktop)

✅ shell 执行结果:
[显示命令输出]
```

### 3. **清空对话**
点击右上角的"清空对话"按钮可以：
- 清除所有聊天记录
- 重置对话上下文
- 开始新的对话

## 🔧 高级配置

### 修改端口

编辑 `webui.py`，找到最后一行：

```python
socketio.run(app, host='0.0.0.0', port=5000, debug=False)
```

修改 `port=5000` 为你想要的端口号。

### 允许远程访问

默认配置已经允许远程访问（`host='0.0.0.0'`）。

要访问，使用服务器的IP地址：
```
http://你的IP地址:5000
```

### 启用调试模式

将 `debug=False` 改为 `debug=True`：

```python
socketio.run(app, host='0.0.0.0', port=5000, debug=True)
```

⚠️ **注意**：调试模式仅用于开发，不要在生产环境使用。

## 🐛 常见问题

### Q1: 无法连接到服务器
**A**: 检查：
1. Web UI是否正在运行
2. 防火墙是否阻止了5000端口
3. 浏览器控制台是否有错误信息

### Q2: 消息发送后没有响应
**A**: 可能原因：
1. LLM API连接问题
2. 检查终端输出的错误信息
3. 刷新页面重新连接

### Q3: 工具执行失败
**A**: 
1. 检查Skill是否正确加载
2. 查看终端中的详细错误信息
3. 确认Shell命令的语法正确

### Q4: 界面样式不正常
**A**: 
1. 检查网络连接（需要加载CDN资源）
2. 尝试清除浏览器缓存
3. 使用现代浏览器（Chrome、Edge、Firefox）

## 📊 技术架构

### 后端
- **Flask** - Web框架
- **Flask-SocketIO** - WebSocket支持
- **JwClaw核心** - Agent逻辑

### 前端
- **Bootstrap 5** - UI框架
- **Socket.IO Client** - WebSocket客户端
- **Marked.js** - Markdown渲染
- **Bootstrap Icons** - 图标库

### 通信
- **WebSocket** - 实时双向通信
- **REST API** - 状态查询
